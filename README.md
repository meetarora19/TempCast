# Temporal Event Prediction from Multimodal Streams

## Overview
This project predicts future events using multimodal data (text + images) and temporal reasoning.

## Features
- Multimodal embeddings (CLIP + SentenceTransformer)
- Temporal-aware retrieval using FAISS
- Cloud-ready API (FastAPI)
- Modular architecture

## Tech Stack
- Python
- PyTorch
- Transformers
- FAISS
- FastAPI
- AWS EC2

## Run
pip install -r requirements.txt
python app.py
